# Year-End Reports Export - Quick Guide

## For E-Resource Editors

### How to Export Reports

#### Step 1: Access the Export Page
1. Log in to CEAL Statistics at https://www.cealstats.org
2. Go to your Admin Dashboard
3. Find the **"Year-End Reports"** card (green icon) in the E-Resource Editor Section
4. Click **"Access"** button

#### Step 2: Select Year
- Use the dropdown menu to select the year you want to export
- Available years: Current year + past 10 years
- Example: Select "2024" for 2024 statistics

#### Step 3: Export Options

##### Option A: Export Individual Form
1. Find the form you need in the list (1-10)
2. Click the **"Export"** button next to the form name
3. Wait for the download to complete (1-2 seconds)
4. File will be saved to your Downloads folder
5. Green checkmark ✓ appears when complete

##### Option B: Export All Forms (Batch)
1. Click the **"Export All Forms (Batch)"** button at the top
2. Wait for the download to complete (5-10 seconds)
3. Single file with all 10 forms will be downloaded
4. All forms will show green checkmarks ✓

### What's Included in Each Export

✅ Institution names (alphabetically sorted)  
✅ All data fields with numbers  
✅ All subtotals and totals  
✅ Notes field (with full UTF-8 support for Chinese/Japanese/Korean characters)  
✅ Professional formatting with headers and borders  
✅ Title row with year information  

### Opening the Files

1. **Microsoft Excel**: Double-click the file (Excel 2016 or later recommended)
2. **Google Sheets**: Upload to Google Drive, then open with Google Sheets
3. **LibreOffice**: Open with LibreOffice Calc

All files use UTF-8 encoding, so international characters will display correctly.

### Troubleshooting

**Problem**: Export button is grayed out  
**Solution**: Make sure you've selected a year from the dropdown first

**Problem**: "No data found" error  
**Solution**: The selected year has no data in the database. Try a different year.

**Problem**: File won't download  
**Solution**: Check your browser's popup blocker settings. Allow downloads from cealstats.org.

**Problem**: Characters look strange in Excel  
**Solution**: Make sure you're using Excel 2016 or later, or try opening in Google Sheets.

### Tips

💡 **Use Batch Export** if you need all forms - it's faster than exporting individually  
💡 **Check the year** before exporting to make sure you're getting the right data  
💡 **Green checkmarks** show which forms you've already exported  
💡 **Files are ready to use** - no additional formatting needed
